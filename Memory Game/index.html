Welcome to Dementia, a take-off of the beloved children's game in which you memorize the placement of cards and attempt to reveal them in matching pairs.

This game is accessible by mouse, keyboard, or touch screen. Keyboard users can use the arrow keys to navigate cards. This game is not currently accessible by screen reader. People with mobility issues may encounter issues as the game progresses.

This game is intended to be a fun way to test your memory skills. There are some obvious ways to cheat but we encourage you to rely on only your memory. Besides, it's a simple child's game. Why would you need to cheat?
